# React Mobile

The boilerplate for rendering native apps on mobile phones using [react-native](https://reactnative.dev/).

## Installation

```bash
npx create-react-prime@latest -b
```

or

```bash
npm init react-prime@latest -b
```

And select **react-mobile**.
